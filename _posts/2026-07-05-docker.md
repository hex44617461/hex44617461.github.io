# 도커|docker 컨테이너화 도구

## 도커 명령어

### docker compose build

docker-compose.yml 또는 Dockerfile 참고해서, 이미지만 생성

### docker compose up --build

docker-compose.yml 또는 Dockerfile 참고해서, 이미지 생성(이미 있다면 덮어쓰기)하고 컨테이너 실행

### docker compose up -d

docker-compose.yml 또는 Dockerfile 참고해서, 백그라운드에서 이미지 생성(이미 있다면 그거 참고)하고 컨테이너 실행

### docker compose down

실행되고 있는 컨테이너 중단하고 삭제, 이미지는 보존
